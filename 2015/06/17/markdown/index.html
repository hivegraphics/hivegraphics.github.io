Markdown语法简介

2015-06-17

Markdown是一种轻量级的标记语言，它使用简易的纯文本格式编写文档，然后再转化成有效的HTML文档。我在第一次使用Markdown时便被她的简洁深深的吸引住了，她让我能够更专注的去编写文档的内容而不需要花太多时间去注意文档的格式。

本文通过整理网上各种资料，总结了Markdown的一些基本语法，用作参考文档。同时，推荐一款在线Markdown编辑网站—作业部落，它提供了很多方便实用的功能和强大的实时预览功能。

1. 斜体和粗体

使用 * 和 ** 表示斜体和粗体。

*斜体*
**粗体**

效果：斜体，粗体

2. 超链接

使用 [描述](链接地址) 为文字增加超链接。

点击[本人博客](http://leechengpeng.com/)可以跳转到我的博客。

效果：点击本人博客可以跳转到我的博客。

3. 分级标题

Markdown有两种类型的标题格式：Setext和atx

3.1 Setext

在文本下一行添加一行”=”表示H1
在文本下一行添加一行”-“表示H2

this is an H1
=============
this is an H2
-------------

效果如下：

this is an H1

this is an H2

3.2 atx

在行首插入1到6个“#”，表示标题的级数

效果如下：

H1

H2

H3

H4

H5
H6

4. 列表

4.1 无序列表

使用 *，+，- 表示无序列表（注：*，+，-与文本之间需添加一个空格）。

效果：

  • Red
  • Green
  • Blue

4.2 有序列表

使用数字和点表示有序列表。

效果：

  1. 有序列表项1
  2. 有序列表项2
  3. 有序列表项3

5. 水平分割线

在空行中输入三个或者更多的 “-“, “*”, “_”即可出现一条水平分割线

* * *

***

*******

- - -

--------------

____

6. 文字引用

使用 > 表示文字引用。

> 野火烧不尽，春风吹又生。

效果如下：

野火烧不尽，春风吹又生。

7. 代码块

7.1 行内代码块

使用形如 `代码` 表示行内代码块。

让我们聊聊 `html`

效果：让我们聊聊 html。

7.2 简要代码块

使用四个缩进空格表示代码块，但需注意缩进空格之前需空一行。

效果如下：

这是一个代码块，此行左侧有四个不可见的空格。

```
